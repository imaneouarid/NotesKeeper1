# NotesKeeper

NotesKeeper is a simple web application for managing notes. It allows users to create, read, update, and delete notes.

## Features

- Create new notes with a title and content.
- View all existing notes.
- Update existing notes.
- Delete notes.

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- Jest (for testing)
